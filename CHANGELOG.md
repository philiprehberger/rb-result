## [0.1.0] - 2026-03-10

### Added
- Initial release
- `Ok` and `Err` result types
- `map`, `flat_map`, `map_err` transformations
- `unwrap!` and `unwrap_or` value extraction
- `Result.try` for exception-safe wrapping
- Ruby 3.1+ pattern matching support (`deconstruct`, `deconstruct_keys`)
